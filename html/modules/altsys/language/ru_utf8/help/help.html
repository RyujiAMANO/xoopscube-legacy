Modules Blocks

Modules can have several predefined blocks that provide specific functionality. These blocks are made available in the block management by activating modules. Each module block has its own settings and template.

Cloning Modules Blocks

Although the system design specification describes the block management process running a single instance of a module block, the Components' module additionally provides a function "force clone" allowing you to duplicate any block.

Force clone creates a copy of a block as it exists at the time of the cloning operation, including the block content and settings.

It is recommended to test the cloned block for any reports of conflicts.

Duplicate Modules Blocks

Perform the following steps to clone a block.

  1. Click the module to list the blocks
  2. Click "force clone" on the block to duplicate
  3. Specify a new block title and settings

This procedure creates an exact copy of the block with the new title and settings.

Language

Components library provides the facility to manage / for multiple languages String Constants: GUI Internationalization

Each language has its own set of files stored in sub-catalogs /module/language/ with translations of language strings constants.

A string constant may consist of any combination of digits, letters, escaped sequences, placeholders and spaces.

You can override a specific language string without modifying any language files.

  1. Select the module to translate
  2. Select the language, e.g. english
  3. Select the file to customize
  4. Add a new Custom Value
  5. Update to save

Language Placeholders

Language constants can contain technical data like HTML markup and placeholders. In most cases these elements should not be modified or translated.

It's important to keep any placeholder defined within a constant, for example, %s or {0}

For example, define( '_MSC_INTSITE' , 'Interesting Site: %s'); here %s is a placeholder for your site name.

Overrides of language strings are stored in your database and saved to a file e.g. /trust_path/cache/lang_english_file-name.php
