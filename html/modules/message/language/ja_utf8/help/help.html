Message

このモジュール Message は、追加機能と拡張機能を備えたプライベート メッセージ機能を提供します。 プライベート メッセージは、特定の受信者のみがメッセージを表示または読み取ることができる通信の一種です。 これは、ダイレクト メッセージ (DM) またはパーソナル メッセージとも呼ばれます。

特徴

  • 受信トレイと送信トレイのメッセージを表示するブロックを追加します。
  • メッセージのステータス、既読、未読、ロック。
  • 受信トレイと送信トレイのステータスまたはユーザー名でメッセージを検索します。
  • メッセージを自動的にパージするオプション。
  • ページネーション (ページあたりのメッセージ数) を定義するための設定。
  • メッセージをメール アカウントに転送する。
  • ブラックリストの管理。

ブロック

登録ユーザー向けに「新しいメッセージ」ブロックをインストールします。

設定

管理者は、1 ページあたりのデフォルトのメッセージ数を設定できます。

メッセージのパージを遅らせ、未読メッセージも削除するかどうかを設定します。

登録ユーザー向けの「新しいメッセージ」ブロックをインストールします。

デフォルトのユーザー プロファイル テンプレートをこのモジュールのテンプレートに置き換えます
「myfriend」モジュールおよび、または「ソーシャル ネットワーク」モジュールをインストールする場合。
