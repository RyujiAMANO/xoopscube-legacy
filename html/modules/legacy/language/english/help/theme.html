The Renderer uses Smarty Template Engine —one of the most popular and mature template engines.

Smarty is a template engine for PHP. More specifically, it facilitates a manageable way to separate application logic and content from its presentation.

This is best described in a situation where the application programmer and the template designer play different roles, or in most cases are not the same person.

The Render engine (renderer) ensures compatibility with legacy themes of XOOPS2 and XOOPS2-JP.

Smarty Template Engine is used for rendering and cache management, it has neither override nor fall-back mechanism.

The renderer uses a theme file "manifesto.ini.php" for specs and details.

The simplest theme requires only one file: theme.html, and optionally style.css to control the style. You can use only theme.html, style.css and manifesto.ini.php to build your theme.

The directory /html/themes/ holds the public default theme.

The directory /html/modules/legacy/admin/theme the admin theme and backend templates.

Themes are built with HTML, CSS, JavScript and Smarty with custom delimiters similar to HTML markup language that make easier to design and develop components for themes and modules templates.

Themes are the skin of XOOPS Cube websites. You may change the look-and-feel of your site by changing the main theme. How to customize the graphical identity your site?

  • You can start by downloading free themes that you can modify.
  • Read the Theme documentation and design your own theme.

How to change the main theme?

The main theme is the default theme of your site.

  1. Click 'Theme management' in the side menu of Legacy module.
  2. Click 'Select' button of the theme that you want to use as the main theme.

How to install new themes?

Download and unzip the theme archive. Then upload the theme folder to the directory "/themes" of your of XOOPS Cube website

Attention !
  • Check that your theme folder includes the following files
    theme.html
    style.css
    manifesto.ini.php.
Renderer

A theme may need a special render-system. 'Renderer' which is released by default, can handle the theme formats of XOOPS2 and XOOPS Cube Legacy. But you can install other render systema and extend XOOPS Cube with themes compatible with touchscreen Mobile devices. Please, read the theme installation note, if any, to avoid issues displaying your site content.

How can users select their favorite Theme?

Users may select their favorite theme from a theme list that you specify. To that end, you need to list up selectable themes and install 'Theme Select' block for visitors.

  1. At first, let's make a theme list. Click 'Theme management' in the side menu of Legacy module.
  2. Check the boxe of each theme that you want to make as selectable. Then, Click button 'Submit'
  3. Install the block 'Theme Select' and set block permissions.
