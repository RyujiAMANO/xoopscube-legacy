Overview

Les préférences système vous permettent de personnaliser les paramètres généraux de votre plate-forme, tels que le nom du site et le slogan, la langue et le thème, activer la recherche globale intégrée, fermer votre site pour maintenance, désigner un module pour votre page d'accueil et bien plus encore.

Panneau de configuration »» Système »» Préférences »» Réglages généraux

Les paramètres de configuration sont agrégés et organisés en plusieurs modules principaux pour garantir la fiabilité, la disponibilité et la possibilité de faire évoluer votre application Web. Les modules de base sont nécessaires pour accomplir les activités d'entrée, de traitement, de sortie, de stockage, de mise à jour et de contrôle. L'accès aux principales fonctions, module utilitaire ou service, de configuration et de gestion sont représentés par les icônes du menu latéral du Panneau de configuration.

Pour en savoir plus, vous pouvez accéder à l'aide intégrée dans le sous-menu de chaque module et activer l'aide contextuelle (panneau d'administration).

Mise en route

Cette section fournit un guide de démarrage rapide avec un bref résumé du panneau de commande, du tableau de bord et de la liste de contrôle post-installation.

Guide de démarrage rapide

Une fois l'assistant d'installation terminé, votre site Web est préconfiguré avec les paramètres de configuration par défaut. Ces paramètres font référence à la valeur préexistante d'un paramètre configurable par l'utilisateur qui est attribué à votre plate-forme d'application Web. Par conséquent, vous devez effectuer certaines tâches de post-installation avant de lancer votre nouveau site Web. Celles-ci incluent la configuration du nom et du slogan du site, la personnalisation des balises méta (SEO), l'autorisation des visiteurs à s'inscrire et les autorisations des groupes d'utilisateurs. Des documents plus détaillés sont disponibles et seront mis en évidence tout au long de ce guide.

Panneau de configuration et tableau de bord personnalisé

Le panneau de contrôle, également appelé zone d'administration, vous donne accès à toutes les actions pouvant être effectuées sur votre plate-forme d'applications Web. Il comprend également un tableau de bord qui peut être personnalisé depuis le "panneau d'administration". Vous pouvez activer ou désactiver les blocs d'administration, les conseils de module et les info-bulles d'action. Vos préférences sont stockées dans le localStorage de votre navigateur.

Menu principal

Le menu principal est généré automatiquement à partir des modules actifs. Chaque sous-menu vous donne accès aux actions du module qui peuvent être effectuées et au Guide de l'utilisateur intégré.

Contenu principal

L'idée principale d'avoir un tableau de bord personnalisable dans XCL est de fournir un accès facile aux différentes sections d'un simple clic. Pour y parvenir, nous avons sélectionné les fonctionnalités clés de XCL que les administrateurs utilisent le plus souvent au quotidien. Nous avons prérempli le tableau de bord avec des "blocs d'administration" par défaut qui vous donneront un aperçu de l'environnement.

Vous pouvez ajouter vos propres widgets au tableau de bord à différentes fins - par exemple, vous pouvez intégrer n'importe quelle API tierce ou widgets de surveillance. En bref, les widgets sont une solution rapide et non technique, tandis que l'API est un processus plus complexe qui nécessite des connaissances techniques spécifiques.

Panneau d'administration

Le x-cube à droite ouvre le panneau d'administration où vous pouvez rechercher une action ou afficher l'aide du module actif, et personnalisez les options de votre tableau de bord.

Liste de vérification post-installation

Une fois l'assistant d'installation terminé, effectuez les tâches suivantes, selon le cas:


Configurer les paramètres

  • Modifier le nom et le slogan du site
  • Adresse e-mail de l'administrateur
  • Message personnalisé de maintenance
  • Sélectionnez "Page de démarrage" ou module
  • Définir le cache à l'échelle du module (production )


Accès utilisateur

  • Autoriser les visiteurs à créer un compte
  • Autoriser le téléchargement d'avatars personnalisés
  • Ajouter de nouveaux champs au profil utilisateur
  • Attribuer des autorisations aux groupes d'utilisateurs
  • Configurer les modules et bloquer l'accès


Préférences de rendu

  • Ajoutez votre identité visuelle
  • Ajouter des mots-clés et description
    selon les besoins de l'entreprise
  • Modifiez la notice de copyright
  • Ajoutez id SEO Google, Facebook, Yandex
L'aide intégrée fournit de la documentation pour aider les administrateurs à gérer l'application web et les modules. Vous trouverez le lien "Aide" dans le sous-menu de chaque module. N'hésitez pas à les consulter.
