Profile Module

The module Profile features a simple user-friendly interface to manage custom fields for the User profile. Administrators can create new user profile fields : checkbox, menu of choices, text area, text input and may be required or not.

New profile fields will appear on each user's profile page only to user groups selected in "Access groups" otherwise it is not visible, in which case only the administrator can see the field. To create a new profile field, select the profile field Type you require from the dropdown list.

You can set the order in which your custom profile fields appear the User profile page.

Custom fields

  • String - allows a series of characters to be stored.
  • Text - textarea with editor.
  • Int (Integers) - allows value containing only decimal digits.
  • Date - allows a specific date format.
  • Checkbox - allows a selection 0 or 1, true or false.
  • Select box - allows multiple options, see the example.
