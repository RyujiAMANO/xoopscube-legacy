Profile Module

The module Profile features a simple user-friendly interface to manage custom fields of User Profile.

Custom fields

  • String - allows a series of characters to be stored.
  • Text - textarea with editor.
  • Int (Integers) - allows value containing only decimal digits.
  • Date - allows a specific date format.
  • Checkbox - allows a selection 0 or 1, true or false.
  • Selectbox - allows multiple options, see the example.

Download Profiles

User data is only available in CSV format order by user_id
