Pre-Built Theme

The render engine is framework-agnostic. This means you can simply "Select" the theme you want to use in Theme Management as the default theme for your website. Any checked theme will also be available in the block "Themes" for user groups with permissions. Users can then select their favorite theme and customize the look and feel of the website.

Templates of Core Modules

Developing a theme takes a little of practice. There are multiple templates for each module. The starter theme can help you to get started through a skeleton layout upon which you can build and design to suit your specific needs. By using a “CSS framework”, optimized for desktop and mobile devices, XCL Theme prioritizes rapid prototyping and quick feedback.

CSS custom properties

Bootstrap 5 is evolving with each release to better utilize CSS variables for global theme styles, individual components, and utilities. It provides variables for colors, font styles at :root level. On components and utilities, CSS variables scoped to CSS class can easily be modified, thus meeting XCL requirements for enhanced real-time customization.

Components and Templates

The XCL Theme starter includes by default the Bootstrap 5 bundle minified and Bootstrap icons. In addition to these default components, the theme also bundles multipurpose features that you can use and reuse as much as you want in your theme, modules and blocks templates.

Color Mode

Bootstrap now supports color modes, or themes, as of v5.3.0. Allowing to switch from default light color to the new dark mode in XCL Theme Starter.

LocalStorage

The currently preferred color-scheme selected by Users will be preserved during navigation by using the Local Storage API.

Block Position Extended

Blocks are used for menus or content. XCL Theme Starter extends the flexible block position system with navbar, bottom and panel.

Navbar Dropdown Blocks

XCL Theme Bootstrap 5 adds a new position block feature with built-in support for dropdown blocks menus inside the main navigation bar.

Bootstrap Icons

Free, high quality, open source icon library with over 1,800 icons. Include them anyway you like—SVGs, SVG sprite, or web fonts. Use them with or without Bootstrap in any project.

Custom Components & Templates

The render engine provides several ways to create and access custom components and named templates. It is flexible enough to include one custom component from within another template. You can also import a Single File Component and all the resources needed in order to output the final layout.

Custom Components are a type of template that are not part of a core module. However, these components can be created with a base amount of Smarty logic that you can insert into your template and global theme.
